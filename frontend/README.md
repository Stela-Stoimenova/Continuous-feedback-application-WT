# Continuous Feedback - Frontend

React frontend application for the Continuous Feedback system.


### 1. Install Dependencies

```bash
cd frontend
npm install
```

### 2. Start Development Server

```bash
npm run dev
```

The application will be available at **http://localhost:3001**

## what you need :
- Node.js 18+ 
- npm or yarn
- Backend server running on port 3000
